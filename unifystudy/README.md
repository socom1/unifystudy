UnifyStudy is an all-in-one productivity platform designed specifically for students. It bridges the gap between fragmented study tools by combining **task management**, **real-time collaboration**, **academic planning**, and **gamified focus tools** into a single, cohesive dashboard.

Whether you're tracking assignments, coordinating with classmates in your university channel, or entering "Zen Mode" for a deep study session, unifystudy has you covered.

## ğŸ Getting Started

Follow these steps to set up the project locally:

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/rejuszuzevicius/unifystudy.git
    cd unifystudy
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    The app will launch at `http://localhost:5173`.

4.  **Build for Desktop (Electron)**
    ```bash
    # For macOS
    npm run build:mac

    # For Windows
    npm run build:win
    ```
